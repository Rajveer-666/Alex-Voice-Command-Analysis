Interpretation
Key Takeaways
Logistic Regression

Best test accuracy (78.65%) and F1-score (80.81%). Strong recall with fewer false negatives than RF. Simple and interpretable — good baseline for production if latency matters.

Random Forest

Best cross-validated accuracy (79.28%) indicating stronger generalization. Most balanced false positive/negative trade-off. Ensemble strength reduces variance despite small dataset.

SVM · RBF Kernel

Highest AUC (0.89) and recall (97.73%) — only 1 missed Alexa in 44. Ideal when missing the wake word is unacceptable, at the cost of more false alarms (21 false positives).
